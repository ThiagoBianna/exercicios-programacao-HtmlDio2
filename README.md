Projeto de um site para a Clínica, criado como desafio prático do Módulo de HTML da DIO. O site possui 4 páginas e aplica conceitos essenciais de HTML5: estruturação com tags semânticas, tabelas de horários, formulário de contato e mídias (iframes e imagens). Também adicionei um CSS básico para deixar o layout limpo e com cara de um projeto real e profissional. Ótimo exercício para fixar os fundamentos do desenvolvimento web!
